# Wikipedia Knowledge Graph

![Screenshot](imgs/screengrab.jpg)

I have been spending a lot of time on [Wikipedia](https://en.wikipedia.org) and I wanted to try [Cosmo.gl](https://github.com/cosmograph-org/) to visualize two entities to see how they are distinct and overlap.

## Stack

- [Cosmos.gl](https://github.com/cosmograph-org/) — GPU-accelerated WebGL graph rendering
- [d3-force](https://github.com/d3/d3-force) — static pre-computed force layout
- React 18 + TypeScript + Vite
- Zustand for state management
